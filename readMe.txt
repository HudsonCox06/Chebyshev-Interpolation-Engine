To run this program, first ensure you have the .jar file contained in the same directory as the project. If not, you must download file yourself and update the commands accordingly.

To compile and run on Windows:
	javac -cp ".;jfreechart-1.5.6.jar;jcommon-1.0.24.jar" Project3.java
	java -cp ".;jfreechart-1.5.6.jar;jcommon-1.0.24.jar" Project3

To compile and run on Mac/Linux:
	javac -cp ".:jfreechart-1.5.6.jar:jcommon-1.0.24.jar" Project3.java
	java -cp ".:jfreechart-1.5.6.jar:jcommon-1.0.24.jar" Project3
